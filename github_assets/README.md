# Repository Assets

This directory contains the visual assets used for the project's documentation, storefront, and main `README.md` file.

## Contents

* **`screenshot.png`**: Static screenshot of the application's user interface.
* **`demo.webp`**: Animated WebP showcasing the real-time waveform visualizer in action.
